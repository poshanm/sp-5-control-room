INSTALL KARNE KA TARIKA:
------------------------
1. Yeh poora folder USB/Pendrive mein copy karo
2. Dusre PC pe copy karo (Desktop ya koi bhi jagah)
3. INSTALL_PA_AI.bat pe RIGHT CLICK karo
4. "Run as administrator" select karo
5. Screen pe instructions follow karo

ZAROORAT:
---------
- Windows 10/11 (64-bit)
- Internet connection (packages download ke liye)
- COM4 port available (relay module ke liye)

INSTALL KE BAAD:
----------------
- C:\PA_AI\master_pa_controller.py mein AZURE_KEY update karo
- C:\PA_AI\telegram_control.py mein TOKEN check karo
- COM port check karo (default: COM4)
